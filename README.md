# Empresa Telefónica En Línea

Proyecto desarrollado para la asignatura **Taller de Integración de Software**.

## Descripción

Esta aplicación web corresponde a un formulario de registro de clientes abonados de la empresa telefónica **En Línea**.

El proyecto fue desarrollado con fines académicos y será utilizado para demostrar la creación de un instalador mediante **Inno Setup**.

## Tecnologías utilizadas

- HTML5
- CSS3
